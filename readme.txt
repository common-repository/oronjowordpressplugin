== Installation ==

__Please switch to our new plugin [Sell Content with Oronjo](https://wordpress.org/plugins/sell-content/) which is 100% compatible with the Oronjo Wordpress Plugin. If you use our new Sell Content plugin, you can safely remove the Oronjo Wordpress Plugin.__

1. Search for 'OronjoWordpressPlugin' in your plugin menu in the Wordpress dashboard, install and activate.
2. Setup the plugin through the 'Settings > Oronjo Plugin' >  menu in Wordpress

or

1. Download the plugin and unzip it.
2. Upload the folder oronjowordpressplugin/ to your /wp-content/plugins/ folder.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Setup the plugin through the 'Settings > Oronjo Plugin' >  menu in Wordpress
5. Installation finished.
